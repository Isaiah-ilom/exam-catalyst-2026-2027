================================================================================
                           WHAT YOU NEED TO DO
================================================================================

STEP 1: INSTALL NODE.JS (REQUIRED!)
------------------------------------

1. Open your web browser
2. Go to: https://nodejs.org/
3. Click the GREEN button that says "LTS"
4. Download and install
5. Restart your computer

That's it! Node.js is now installed.


STEP 2: INSTALL PROJECT DEPENDENCIES
-------------------------------------

Double-click this file:
   install-dependencies.bat

Wait 5-10 minutes for it to complete.


STEP 3: RUN THE APPLICATION
----------------------------

Double-click these files (in order):
   1. start-backend.bat
   2. start-frontend.bat

Your browser will open automatically!


STEP 4: USE THE APP
-------------------

1. Register an account
2. Login
3. Start practicing with 15 UTME subjects!

================================================================================
                           TROUBLESHOOTING
================================================================================

Problem: "npm is not recognized"
Solution: Install Node.js from https://nodejs.org/

Problem: Installation is slow
Solution: This is normal. Wait 5-10 minutes.

Problem: Port already in use
Solution: Type Y when asked to use different port.
